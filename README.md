# copy-to-open
Copy a path, no matter it's 

`c:\windows\abc.txt` 

or

`c:/windows/a.txt`

 even `c:/windows\abc.txt`,
then `ctrl+shift+O`, it opened. 

Use `ctrl+shift+Q` to ask google!

So easy!
